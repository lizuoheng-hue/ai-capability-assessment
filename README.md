# AI 能力等级测评

一个面向职场用户、学生、内容创作者和 AI 学习者的纯前端能力测评工具。通过 15 道行为情境题，从 5 个维度评估用户使用 AI 的能力，并生成等级、雷达图和针对性的提升建议。

## 在线体验

[https://ai-capability-assessment-cn.pages.dev/](https://ai-capability-assessment-cn.pages.dev/)

## 功能特点

- 15 道单题聚焦式情境题，约 6 分钟完成
- 自动保存当前选择，支持返回修改和重新开始
- 实时展示五维能力分布与预估分数
- 根据本地规则生成总分和 L1-L6 能力等级
- 结果页包含优势、短板和下一阶段训练建议
- 支持打印或保存结果页
- 响应式布局，适配桌面端和移动端
- 无需登录，不接入后端，不上传用户答题数据

## 测评维度

| 维度 | 说明 |
| --- | --- |
| AI 基础认知 | 理解模型能力、限制、幻觉和上下文 |
| 提示词表达 | 拆解任务、提供约束和示例的能力 |
| 工具使用 | 使用通用 AI、编程和自动化工具的熟练度 |
| 工作流整合 | 将 AI 融入写作、分析、代码和运营流程的能力 |
| 判断与校验 | 验证来源、识别错误及处理隐私风险的能力 |

## 等级体系

| 等级 | 名称 | 描述 |
| --- | --- | --- |
| L1 | 旁观者 | 听说过 AI，但很少使用 |
| L2 | 尝鲜者 | 能完成简单问答和内容生成 |
| L3 | 助手型用户 | 能用 AI 提升单点任务效率 |
| L4 | 协作型用户 | 能与 AI 多轮协作完成复杂任务 |
| L5 | 工作流设计者 | 能把 AI 融入稳定工作流程 |
| L6 | AI 原生专家 | 能设计自动化、评估质量并指导他人 |

## 本地运行

项目不需要安装依赖，可以直接打开 `index.html`，也可以启动一个本地静态服务器：

```bash
python3 -m http.server 4173
```

然后访问：

```text
http://localhost:4173
```

## 评分方式

每道题包含 4 个行为描述选项，对应不同能力成熟度。每个维度包含 3 道题，维度得分会归一化为 0-100 分；总分由五个维度等权计算，再映射到 L1-L6 等级。所有计算均在浏览器本地完成。

## 隐私说明

当前版本不包含账号系统、数据库或第三方分析脚本。用户答案仅保存在当前页面运行状态中，刷新或重新测评后会清除。
